How this atlas is structured

Short orientation for MSc/PhD users building hypotheses from this view.

Wild-type expression data, disease orthology, and GO annotations are exported from ZFIN and collated into a single Excel file. This interface lets you slice that file by gene, stage, anatomy, disease, and GO term. For full curation context, always follow through to ZFIN.
